Running live at: http://apps.baljinderbains.com/

To run locally import the calendar database
and grant root user (with no password) all privileges to it.

Alternatively, edit the credentials in php_includes/db_connect.php
to your preferred credentials.

Project details:

- Allows users to create multiple todos on a single day.
- Allows users to delete todos.
- Allows users to edit todos.
- Responsive design.
- Implemented without pre-existing frameworks, libraries or widgets.
- Allows users to navigate to earlier and later months.
- Prevents users from entering todos for dates in the past.

Approximately 300 lines of code.
